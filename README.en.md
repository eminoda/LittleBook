# LittleBookShop

#### Description
LittleBookShop（小人书） 一个综合的前端 Node.js 项目，用来展示连环画书籍的店铺。技术栈涉及：Vue+Vant+Egg+Mongo，你可以作为前端（全栈）开发的练手入门项目。
