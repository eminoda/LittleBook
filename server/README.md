# Node 服务端-基于 Egg

## 快速上手

```shell
# 进到服务端项目文件夹
cd GitProject/server/

# 安装 npm 依赖
npm i

# 启动开发模式
npm run dev

# 访问链接： http://localhost:7001/
```
